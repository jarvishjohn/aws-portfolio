# aws-portfolio
My AWS-hosted portfolio website
AWS Static Portfolio Website

This project is a responsive portfolio website built using HTML, CSS, and basic JavaScript.

Deployment Details:
- Hosted on Amazon S3
- Static website hosting enabled
- Public access configured using bucket policy
- Website accessible via S3 website endpoint

AWS Services Used:
- Amazon S3
- IAM (Bucket Policy)
